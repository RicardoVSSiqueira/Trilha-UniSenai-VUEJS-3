# Trilha 02 — Conhecendo o Vue.js

## Estrutura de pastas
```
OT02_ConhecendoVue/
├── respostas-teoricas.md   # Parte 1 - 15 questões teóricas
└── index.html               # Parte 2 - 10 questões práticas (todas em um único arquivo, por seção)
```

## Como rodar
Basta abrir o `index.html` direto no navegador (duplo clique, ou botão direito → "Open with Live Server" no VS Code).
Não precisa de backend nem de instalação — tudo roda via Vue 3 pelo CDN.

## Exercício 1 (jsFiddle)
O código dentro da seção "Exercício 1" do `index.html` é exatamente o que você pode colar
no jsFiddle: HTML na aba HTML, o `<script>` da instância Vue na aba JavaScript
(sem a tag `<script>`), e adicionar `https://unpkg.com/vue@3` em Settings → External Resources.
